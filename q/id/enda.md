# Enda

---

Misalkan anda diberikan dua array yang berisi integer sebagai input, a dan b

*contoh*
```js
    a = [1,3,5,7,9];
    b = [1,2,3,4];
```

## Task

Buatlah sebuah array yang berisi elemen yang hanya berada di a atau b, tapi tidak keduanya

contoh output:

```[2,5,7,8]```
